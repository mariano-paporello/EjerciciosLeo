```http
http://localhost:8080/
```

## Endpoints of the Exercises

First exercise
**Direction normalizer**
EX:
```http
http://localhost:8080/api/normalize
```
---

Second exercise
**Distance calculation**
EX:
```http
http://localhost:8080/api/distanceCalculator/
```

---

Third exercise
**Obtain name of currency**
EX:
```http
http://localhost:8080/api/currencyFinder/
```

## Prerequisites

- node >=16.14.2
- npm >=8.19.2

<br />

## Install

```sh
npm install
```

## Build

```sh
npm run build
```

## Usage

```sh
npm run start
```

## Run as dev

```sh
npm run start:dev
```
